# Single

一个简洁大气，含夜间模式的 Typecho 博客模板，现已移植到 [SilverBlog](https://silverblog.org)。

## 使用方法

```bash
cd templates
curl https://raw.githubusercontent.com/SilverBlogTeam/single/master/install.sh | bash
```
